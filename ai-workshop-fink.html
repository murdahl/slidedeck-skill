Hooks

Eksekver kode automatisk etter Claude-handlinger

// settings.json — Kjør Prettier etter fillagring
{
  "hooks": {
    "afterWrite": [
      {
        "command": "npx prettier --write $FILE",
        "description": "Format med Prettier"
      }
    ]
  }
}

Claude Code — Triks

  • Sett opp CLAUDE.md og settings.json i ~/.claude/
  • Bruk Claude i et prosjekt en stund, så be den generere CLAUDE.md
  • Den lager en baseline basert på dine preferanser
Enkleste måten å starte: kjør claude init i prosjektet ditt
