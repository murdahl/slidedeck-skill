# Dark Theme (Default)

This is the default theme. The CSS variables below are already set in the base `:root` block — no overrides needed.

```css
:root {
  --color-bg: #0f172a;
  --color-surface: #1e293b;
  --color-primary: #3b82f6;
  --color-primary-light: #60a5fa;
  --color-accent: #8b5cf6;
  --color-text: #f1f5f9;
  --color-text-muted: #94a3b8;
  --color-border: #334155;
}
```
